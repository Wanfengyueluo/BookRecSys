# Book-Web

> node 14.7.0
>
> npm 6.14.7

## 依赖下载

进入 `book-web`目录

`npm install`

修改`main.js`中的请求地址，改成Book-Server的地址即可

```js
axios.defaults.baseURL = "http://192.168.2.2:9090/"
```

启动服务

`npm run serve`
